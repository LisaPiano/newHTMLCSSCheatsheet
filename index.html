How to create a table using HTML markup cheatsheet

CheatsheetTable

Tags

Tag Meaning

Tag Role

<table>

Defines table

Add to beginning and end of table

<thead>

Defines heading row(s)

Include all heading titles

<tr>

Defines every row in every catagory

Nests the heading and data

<th>

Defines every heading

Nested inside <tr>, no <td> here

<td>

Data nested inside non-heading row

Use nested inside of <body> and <tr>

<h2>

Good size for data

Use nested inside of <td> or <th>

<tbody>

Delineates data rows in body of table

Nests the data rows

<footer>

Places footer at bottom of table

Contains footer data
